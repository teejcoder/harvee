# Decisions

Short ADR-style notes on material design decisions. One file per decision.

Format: `NNN-slug.md`, numbered in the order they were made. Each ADR captures **context**, **decision**, and **consequences** in a few paragraphs — no template ceremony.

No ADRs recorded yet. Add one when:

- Something on the "deliberately NOT in the stack" list gets added (`.memory/tech-stack.md`).
- A destructive migration is introduced (`.memory/conventions.md` §7).
- A state or transition is added, renamed, or removed (`.memory/state-transitions.md`).
- Any decision reversed by later work — record the original alongside the reversal.
